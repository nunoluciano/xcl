Ajax Content

Fetch external content via Ajax for the tabs by setting an href value in the tab links.

Tabs UI Typo and UI Spinner demonstrate loading components and last Tab broken AJAX, and how to handle serverside errors in those cases.

Note: These two require a webserver to interpret PHP. They won't work from the filesystem.
