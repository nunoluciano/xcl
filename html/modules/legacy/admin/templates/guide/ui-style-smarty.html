Smarty

These smarty variables can be used in the Admin Theme. A theme is a collection of single template files (HTML, CSS, JS, Smarty).

XCL uses custom left <{ and right }> delimiters. Every module can have its own templates and smarty variables. Briefly, you have different smarty variables for the front-end and back-end.

You can also use <{$smarty.const.NAME}> to access PHP constants. And use <{assign}> for assigning template variables during the execution of a template. The scope of the assigned variable can be : 'parent', 'root' or 'global'. Assignment of variables in-template is essentially placing application logic into the presentation that may be better handled in PHP. Use at your own discretion.
