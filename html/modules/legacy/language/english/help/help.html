XCL 2.3.0 KICK START GUIDE

A kick start guide to XOOPSCube XCL if you are running this admin dashboard for the first time.
The goal of this kick-start guide is to help new site administrators get orientated to what this position does and what responsibilities it entails. It provides the minimum amount of information needed, to get started and begin working with XCL Web Application Platform. Other detailed instructional materials are available and will be highlighted throughout.

Read this guide!

Outline

  • This XOOPSCube version XCL 2.3.0 ensures compatibility with Legacy modules, OOP or Object Oriented Programming, class abstraction and interfaces.
  • In 2017 major PHP projects started bug fixing and code refactoring to PHP 7.
  • In computer programming and software design, code refactoring is the process of restructuring existing computer code—changing the factoring—without changing its external behavior.
  • Refactoring is intended to improve the design, structure, and implementation of the software, while preserving its functionality.
  • Moreover, the design and code refactoring of XCL primarily provides backward compatibility and updates modules based on Duplicatable V3 (D3) architecture.
  • Content creation, filemanager with cloud storage management, WYSIWYG editor and X-Update to get and deploy from a simple user interface.
  • You can find out more information in Help menu link about features like wizard to install modules, related menus, localization, permissions and the theme engine.
